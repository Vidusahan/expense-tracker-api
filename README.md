# Expense Tracker API

A REST API for managing personal expenses.

## Features

- User registration
- User login
- Add expenses
- View expenses
- Update expenses
- Delete expenses
- Expense filtering

## Tech Stack

- Python
- FastAPI
- SQLAlchemy
- SQLite
- JWT Authentication

## Project Goal

This project is built to learn:

- REST API development
- Database handling
- Authentication
- Git & GitHub workflow
